Applicatie Functies - gecategoriseerd ()
Om tot een eerste indeling van applicatiefuncties te komen kijk ik naar de bestaande indeling van generieke functies. Omdat deze indeling naar verwachting niet de hele lading zal dekken, vul ik deze mogelijk aan. Van niet alle generieke functies kan ik een applicatiefunctie maken, deze zal ik kleurloos weergeven. Doel van deze view is een overzicht van applicatie functies, ingedeeld op overenkomstige kenmerken.
