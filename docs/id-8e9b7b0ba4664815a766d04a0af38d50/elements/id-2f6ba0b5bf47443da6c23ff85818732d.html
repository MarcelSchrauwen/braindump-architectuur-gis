Veronderstelde toestemmingscontrole functie ()
Grondslag controle functie voor veronderstelde toestemming onder Wgbo. Doel - Duidelijke toetsingscriteria voor systemen die werken onder de grondslag veronderstelde toestemming. Belang - Vertrouwen cliënt en zorgverlener.
