Zorgverlener onweerlegbaarheid functie ()
Het kunnen aantonen dat een specifieke gegevensverwerking daadwerkelijk door de zorgverlener is verricht, zodanig dat de zorgverlener dit achteraf niet geloofwaardig kan ontkennen. Toelichting EGIZ Werkgroep: Het digitaal vastleggen van een keuze van een zorgverlener welke invloed heeft op de gezondheid van een cliënt zou onweerlegbaar vastgelegd moeten worden wanneer deze keuze gedeeld wordt over zorgorganisaties heen. Denk bijvoorbeeld aan het uitschrijven van een recept welke verstuurd wordt aan de apotheek. Momenteel kan alleen UZI dit tot zekere hoogte leveren (de gebruiker ziet echter niet wat hij ondertekent). Onweerlegbaarheid kan ook een grote rol spelen in het machtigen van collega’s (mandatering), het aanwijzen van zorgverleners in het behandelteam en het identificeren
